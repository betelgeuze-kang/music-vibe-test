다시 확인할 수 있는 증빙

  • 공개 권리 manifest JSON
  • 오디오 생성 소스 코드
  • 활성 audioClipId는 소스 registry와 권리 manifest 양쪽에 있어야 하며, 하나라도 빠지면 CI와 런타임 재생이 모두 거절합니다.
  • 과거의 제3자·미검증 MP3는 활성 트리에서 퇴역한 상태입니다.

상업곡 추천은 다른 방식입니다

추천 목록에는 사실적인 곡 정보, 이곳에서 직접 쓴 짧은 설명, Spotify·YouTube·Apple Music 외부 링크만 표시합니다. 상업 음원 파일·가사·앨범 커버를 이 서버에 복제하지 않으며, 아티스트·레이블·플랫폼의 제휴나 승인을 받은 것처럼 말하지 않습니다.
